haneWIN Privat

haneWIN Privat ist eine Java MIDP 1.0 Implementierung zur Verschlüsselung von Einträgen in einer sicheren Datenbank auf einem Mobiltelefon mit Java-MIDP-Implementierung. haneWIN Privat ermöglicht damit die sichere Speicherung von vertraulichen Daten, Passworten und PIN Codes.

  • Zur Überprüfung des Master-Kennwortes wird die MD5-Hashfunktion verwendet. Das Master-Kennwort selbst wird nicht gespeichert.
  • Die Verschlüsselung der Einträge erfolgt mit dem RC4 Algorithmus.
  • Durch die Beschränkung auf MIDP 1.0 ist die Software auf jedem Mobiltelefon mit Java-Implementierung einsetzbar.
  • Speicherung der Einträge auf zwei Hierachieebenen.
  • Import der verschlüsselten Daten über Netz oder wenn unterstützt aus einer Datei auf dem lokale Dateisystem.
  • Export der verschlüsselten Daten.
  • Version mit deutschen oder englischen Texten.
  • Windows Programm mit Quelltext zur Ver-/Entschlüsselung der Daten unter Windows.
  • Übernahme der Einträge aus einer Datei (Works/Excel Tabelle mit Speicherung als *.txt) (unter Windows verschlüsselt oder unverschlüsselt).

Die Software kann kostenlos genutzt werden.
