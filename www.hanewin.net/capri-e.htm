Internet Dialer, Connection Sharing via ISDN
for Windows

The ISDN CAPI 2.0 IP-Routing-Interface (CAPRI32) is a flexible dialer for Internet via ISDN. It operates completly without RAS and supports shared access to the Internet for connections with only one official IP address.

The software was developed for use with ISDN CAPI 2.0 compliant ISDN cards. The software supports:

  • DHCP for automatic client configuration
  • Remote monitor/control program
  • various encapsulation protocols
  • Callback, incoming and outgoing
  • MultiLink-PPP or Loadsharing
  • Data compression, TCP header compression
  • IP masquerading
  • Firewall
  • fast connection set up (without Windows RAS)

The software is Shareware.
