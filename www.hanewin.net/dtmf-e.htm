ISDN DTMF Batch
für Windows

Viele Serviceangebote über Telefon arbeiten mit Abfragesystemen über DTMF (Tonwahl) Sequenzen. Zahlreiche, immer wiederkehrende Nutzungen werden dabei inzwischen vom Anbieter vollständig automatisch ohne Call-Center Mitarbeiter abgewickelt, z.B. Kontostandabfrage, Phone-Brokerage, Kreditkartenprüfung ....
Damit ist aber auch von der Kundenseite eine Automatisierung möglich.

Die ISDN DTMF Batch Software ermöglicht die automatisierte Ausgabe von DTMF Sequenzen über ISDN. Bis zu 5 veränderliche Werte können vor der Ausführung abgefragt werden. Falls erforderlich, kann der Automat auch jederzeit angehalten werden und die weitere DTMF Eingabe manuell über die Tastatur fortgesetzt werden. Die Sprachausgabe der Gegenstelle wird vorgelesen und kann zusätzlich abgespeichert werden. ISDN DTMF Batch ist auf allen Windows Plattformen ab Win9x einsetzbar.

ISDN DTMF Batch ist Shareware.
