Internet Connection Sharing over DSL
for Windows

The PPPoverEthernet IP Routing Interface (POETRI) offers secure access to the Internet over DSL, including shared access for a LAN with only one official IP-address. POETRI emulates a LAN interface, it operates without RAS. The built-in firewall protects your system from unauthorized access from the Internet.

POETRI makes use of the IP routing capabilities of Windows and implements IP masquerading to offer transparent Internet access for a LAN.

The following features are supported:

  • Transfer speed adjustment for asymmetric links
  • DHCP for automatic client configuration
  • Remote monitor/control program
  • PPP with PAP or CHAP authentication
  • Data compression, TCP header compression
  • IP masquerading
  • DNS redirection and DNS interception
  • Firewall
  • Shorthold, to optimize connection time and costs
  • fast connection set up (without Windows RAS)

The software is Shareware.
