haneWIN LLDP Agent
for Windows

The haneWIN LLDP software implements a LLDP Agent based on the IEEE 802.1AB standard for the Link Layer Discovery Protocol.

The Link Layer Discovery Protocol (LLDP) is a protocol for discovering the physical topology in 802 LANs. Adjacent stations are discovered and stored by the LLDP agent for retrieval by a SNMP based network management system.

The LLDP agent is implemented as native service for background operation on Windows 2000 and higher platforms. A Control Panel applet provides interactive access to the service. The LLDP MIB is implemented as SNMP MIB subagent for the Windows SNMP agent.

Control panel user interface in English and German.

The software is shareware.

Further information about the LLDP protocol is available here:
