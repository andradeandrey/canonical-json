Syslog Daemon
for Windows

A syslogd implemented as a Win32 console application.

  • The program can log the entries on screen and/or to a file.
  • The program uses multithreading for independant capturing and logging the data. The size of the capture buffer is configurable, using a default of 128 kBytes.
  • Integrated string match/exclude for selective logging.

The software can be used free of charge.
