PPPoverEthernet DOS Packet Driver for
High-Speed Internet access over DSL

PEPA is a Packet Driver for PPPoverEthernet (RFC2516), the access protocol of DSL.

PEPA was developed for Internet access over DSL. It emulates a standard Ethernet Packet Driver interface to the applications. It has all features for setting up a low cost Internet access router with standard PC hardware.

PEPA offers the following features:

  • PPP with PAP or CHAP authentication
  • TCP header compression
  • IP Masquerading
  • DNS redirection and DNS interception
  • Shorthold, to optimize connection time and costs

The software is Shareware.

Download PEPA 1.9

Setup of an DSL Internet access router requires:

  • a standard PC (486 or higher) with DOS
  • an Ethernet card with an Ethernet Packet driver
  • the PPPoE Packet Driver PEPA
  • and an IP router software like ka9q NOS, IProute, PCROUTE.

Packet and NDIS-2 drivers for DOS based
Internet access via ISDN

The software was developed for use with ISDN CAPI 2.0 or CAPI 1.1 compliant ISDN cards. All drivers emulate an Ethernet LAN interface to the upper layer. The Packet Drivers follow the FTP Packet Driver standard and the NDIS Drivers are based on the Microsoft NDIS specification. An ODI interface can be created by using PDETHER (freeware from Novell) on top of one of the Packet drivers.
All versions support:
  • various encapsulation protocols
  • Callback, incoming and outgoing
  • MultiLink-PPP or Loadsharing
  • TCP-header-compression
  • fast, automatic connection set up
The DOS Packet drivers for use with TSR-or VxD CAPI implementations are available in four flavours:
  • standard version
  • Plus version with CHAP, vJHC and NAT
  • Mega version with IP-Masquerading and Name resolver. For dial out IP routers from LANs with only one official IP address
  • 8B version supporting up to 8 B-channels for dial in IP router solutions.
