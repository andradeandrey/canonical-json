haneWIN PPPoE Relay
for Windows

The haneWIN PPPoE Relay software implements a forwarding service for PPPoE packets.

  • Can make an internal DSL card/USB connected DSL modem with PPPoE access available to other computers on a LAN without putting the whole LAN online.
    e.g.

    Computer-1 with DSL Card <--Ethernet/WLAN--> Computer-2...

    Gives Computer-2 Internet access over DSL without putting Computer-1 online. IP traffic is transfered PPPoE encapsulated through Computer-1. Computer-1 is not an IP endpoint.

  • Can make a DSL modem available over WLAN to create your own hot spot.
    e.g.

    DSL Modem <-Ethernet--> Relay Computer <--WLAN--> Computer-2,3,...
    or
    Computer-1 with DSL Card <--WLAN--> Computer-2,3,...

    Gives WLAN clients separate Internet access over DSL. All clients share a common DSL connection, but each client must login at the DSL provider.

The computer running the relay software acts as a forwarder for PPPoE Packets. Only PPPoE packets are forwarded between clients. The interfaces don't need to have the IP protocol activated or an IP address to operate the PPPoE relay.

The haneWIN PPPoE Relay is implemented as native service for background operation on Windows 2000/XP and higher platforms. A Control Panel applet provides interactive access to the service.

The software is shareware.
