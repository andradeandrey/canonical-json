Send Short Messages via ISDN or Modem
from Windows 9x/ME/NT/2000/XP

The software is configurable for SMS center with TAP or UCP protocol offering modem or ISDN X.75, V.120 access.
It supports automatic provider selection and an optional LCR selection for cost optimization.
Automatic provider selection is preconfigured for mobile providers in Germany: ISDN (D1, D2, E+) und modem (O2).
The software runs on all Windows platforms.

Version 1.2:

  • Implemenation of Swiss SMS over ISDN protocol.

Version 1.1 supports:

  • Sending of multiple messages per connection.
  • Sending of preconfigured messages from a file (Batch Mode).
  • Sending of long messages of up to 640 Bytes.

Other features of the software:
  • ISDN call monitor with country/area code database.
    Shows called and calling number and country/area by name (Germany only).
  • ISDN Fax Server. The built-in fax server can store received faxes in standard TIFF format.
    Immediate print out with optional filtering of spam faxes.
  • ISDN answering machine with DTMF receiver.
  • Post processing: After a call, a received fax, a saved voice message or accepted dtmf codes a program can be started for post processing the incoming call (e.g. forwarding the fax or message by email, storing the DTMF codes in a database).

WinSMS is Shareware.
Registered users of ISLA can use WinSMS lite (no Batch-Mode) free of charge on all ISLA clients for use over an ISLA server.
